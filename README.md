# PunnettPy

PunnettPy is a collaborative Python package that simulates Mendelian inheritance by generating Punnett squares and calculating genotype and phenotype probabilities. Users can define custom genetic traits, specify parent genotypes, and automatically compute inheritance outcomes through an easy-to-use object-oriented interface.

Designed for students, educators, and anyone interested in genetics, the package emphasizes modular software design, reusable code, and accessibility for users without a biology background.

## My Contribution

This repository showcases my contribution to the project: the **`traits_class`** class. This class is responsible for:

* Storing genetic trait information and alleles.
* Validating user-provided genotypes.
* Converting genotypes into phenotypes using configurable dominance rules.
* Providing the foundation used by the `PunnettSquare` and `GeneticAnalyzer` classes.

The complete PunnettPy package was developed collaboratively by **Hadiya Cabdi**, **Hodan Ali**, and **Lindsay Kowal**. While the full application consists of multiple interconnected classes, this repository contains only the code I personally implemented to accurately represent my individual contribution.

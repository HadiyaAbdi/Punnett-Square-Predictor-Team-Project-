# Punnett-Square-Predictor-Team-Project-

Punnet square predictor is a collaborative object-oriented Python project that simulates Mendelian inheritance by generating Punner squares and predicting offspring genotype and phenotype probabilities. The project was designed as a reusable genetics package for students and educators while demonstrating object-oriented programming, algorithm design, and software engineering principles.

## My Contribution

This repository contains my implementation of the `Trait` class, which validates genotypes, stores genetic trait information, and converts genotypes into phenotypes using configurable dominance rules. The complete application also includes the `PunnettSquare` and `GeneticAnalyzer` classes developed collaboratively by the project team.
